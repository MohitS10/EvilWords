#EvilWords
A tiny program that reminds you when you are about to type words you should not :)

##Platform
- Linux (Ubuntu)    : READY
- Windows           : Comming Soon

##Commands
- 'close'           : Close Program
- 'add _word_>'     : Add word to dicitionary
- 'enable _word_'   : unmute word
- 'disable _word_'  : mute word
